## Getting Started

Prerequisites:

Make sure you are using Node version 18.x or later.

First, make sure you have the packages you need installed:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

And that's it!
